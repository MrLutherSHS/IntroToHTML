According to the OCR mark scheme:
JavaScript is a programming language that runs in a web browser that can be embedded into HTML with <script> tag to add interactivity to a page.
You need to learn how to follow and write basic JavaScript code. You don't need to memorise exact syntax and can make minor mistakes. You should know how the basic structures work. You don't need to know about JS OOP or file handling.
